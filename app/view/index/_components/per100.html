代替vue的过滤器的wirte、read方式，初始化

因为avalon的xmp组件写法，不能嵌套。所以采用指令来实现。不嫌烦，可以用append也是一种思路

html:
js: 

	{
		$id : "demo_per100",
		aa:"1",
		cc:'{"v":"@aa","w":"@writ($event)"}',
		writ:function(event){
			this.aa=event.target.value/100;
		},
		percentage: '{"v":"x.Percentage","r":"@rad(x)","w":"@writ($event,x)","theme":"nobg text-right"}'//这样的都可以
	}
结果: 一个显示的值是传入的值100倍。（费用分摊百分比用的就是此）
